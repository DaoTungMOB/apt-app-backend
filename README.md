# APT-APP

## Các bước cài đặt

### 1. Cài đặt nvm

Link: <https://github.com/coreybutler/nvm-windows/releases>

### 2. Cài đặt phiên bản nodejs (sử dụng CMD)

```bash
nvm install 20.13.1
nvm use 20.13.1
```

### 3. Cài đặt yarn

```bash
npm i -g yarn
```

### 4. Cài đặt dependencies

```bash
yarn install
```

### 5. Chạy dự án

```bash
yarn start
```

### Dự án sẽ chạy với địa chỉ <http://localhost:8017>
